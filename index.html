JavaScript VideoPlayer

Características

AutoStop

Activando esta opción, cada vez que hagas scroll el video se pausará automáticamente si este está menos del 40% visible y se reanudará si este entra en el rango nuevamente.

Activando esta opción, cada vez que cambies de pestaña, el video se pausará automáticamente.

AutoPlay

Activando esta opción, cada vez que abras esta página, el video comenzará a reproducirse automáticamente, pero sin sonido. (recarga la página para ver los cambios).

Repositorio

Lugar vacio para que puedas scrollear y probar las funciones
